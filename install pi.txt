How to Install teensy loader inside raspberry pi.

copy the file: 
00-teensy.rules et teensy_raspberrypi.tar.gz
from piardumower folder to Home/pi/Documents/

Inside terminal use these command to unpack the 3 files and set the rules

cd Documents/PiArdumower
sudo cp 00-teensy.rules /etc/udev/rules.d/
tar -xvzf teensy_raspberrypi.tar.gz

Now to launch the loader:

./teensy &
